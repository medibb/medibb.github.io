2 근골격계 환자교육

근골격계 외래초진환자는 대부분 잘못된 자세습관, 운동방법이 몸에 배어있다.

때문에 척추와 어깨, 무릎 관절 등에 대한 생체역학 지식에 기반한 올바른 자세와 일상생활 동작의 습관화, 즉 행동치료가 필요하다.

물리치료실에서 시행하는 운동치료에 선행/병행되어야 한다.

각 소주제에 대한 교육용 리플렛 8개를 홍보실에서 제작해서 활용.

  • 오십견
  • 회전근개 파열
  • 무릎통증
  • 팔꿈치 통증
  • 아킬레스, 족저근막염
  • 시술 설명
  • ADL education
    • Lordosis
    • Sagittal alinement
    • Pelvic tilt / Hip hinge
    • 의자 / 침대 / 승하차 / 물건들기 교육
  • Lumbar MKz exercise
    • prone / sitting / standing
  • 이 후 운동프로그램은 PT실에서 진행 예정
  • ADL education

3 척추 인터벤션

3.1 시술 전 준비

화요일과 금요일 오전 (VFSS 다음)

  • 외래환자
    1. 시술 전날, 낮병동(A동 2층) 입원 예정 환자의 CP등록 및 처방
      • 교수님별 CP처방 수정
        • pf. 이재현 : 1레벨에 Dexa 2개(따로 지시 있으면 Tamceton 1개), Hirax 안씀, Skin protect 안 씀
        • pf. 이병주 :
    2. 시술 당일, 환자가 낮병동 입원하면 동의서획득 및 시술 처방
      • 동의서 : “인터벤션 및 조영제사용 동의서”
      • 시술 처방 : “RM S.T.E” 처방 중 부위와 갯수 선택
    3. 투시실(A동 1층, 7362)에 연락해서 시간 확인 후 낮병동에 알려준다.
  • 입원환자
    1. 재활의학과 및 타과 협진환자의 경우, 시술 전날 set처방을 교수님 별로 수정 처방
    2. 시술 전까지 동의서 획득 및 “RM S.T.E” 처방

3.2 시술 보조

  1. 침대 및 환자 이동

    • 시술레벨 아래에 베개 받치도록)
    • 산소 및 혈압 모니터 연결, 혈압측정

  2. 주사 전 상차림 보조

    • 교수님별 Regimen
      • pf. 이재현 :
        • Skin Anesthesia : 3cc syringe + 25G needle
          • 2% lidocaine 1cc + N/S 1cc
        • Medication : 10cc syringe + 22G spinal needle (경추는 25G)
          • lumbar : ropivacaine 0.5cc, normal saline 3cc, tam 1v (dexa 2@ for complicated case)
          • cervical : normal saline 2cc, dexa 2@
        • Contrast : 5cc syringe + extension line
        • 18G needle : 빠른 제조 위해서
      • pf. 이병주 :

  3. 주사 중 환자 확인

    • 경추의 경우 리도카인 테스트 & 스테로이드 주입할 때 반응 확인
      • “주먹을 쥐었다 폈다 해보세요”
      • “발목을 까딱까딱해보세요”
      • “숨이차거나, 이상한 맛이 나지는 않나요?”

  4. 주사 후 혈압 재측정, 메딕스 드레싱, N/S으로 주변 소독약 닦아내기

  5. 환자 이동, pain provocation 위치, 양상을 확인

3.3 시술 후 기록

  1. ‘시술기록 - 공통’ / ‘당일 입퇴원 기록’ 임시저장
    • 예문 활용
    • ‘시술기록 - 공통’ 에 pain provocation 위치, 시술 전 후 혈압 기록
    • ’당일 입퇴원 기록’에는 PACS에 올라온 시술 마지막 사진 붙여넣기
  2. 퇴원지시
    • 시술 경과 평가 위해 3주 후 외래 f/u
    • 퇴원약 처방
      • NSAIDs / PPI 7일
      • 환자 주사 시 pain-provocation 심했거나, peri-neural injection 양상 이었거나, 환자가 그냥 원하는 경우

4 초음파 인터벤션

4.1 검사 전 세팅

  • 초음파 기기 준비
    • ‘초음파(ultrasound)’ 처방만 있었으면, ‘관절 초음파’ 처방 등록
    • 환자 선택 및 검사 시작
    • 기본 프로브 선택
      • 12Hz 이상 Linear probe & MSK 세팅 / gel 도포
      • 고관절 및 척추는 curved probe 선택, Depth 6cm 로 조정
  • 환자 포지션
    • 어깨 : 민소매 환자복으로 환복. 초음파 앞 둥근 의자에 앉아 무릎위에 손바닥이 보이게 올려놓음
    • 무릎 : 바지가 안올라가면 환복. 검사하려는 무릎이 침대 바깥쪽에 오게 누워서, 무릎 아래 배게 받침
    • 팔꿈치 : 환자를 둥근의자에 앉히고 작은 테이블 위에 팔꿈치를 90도 굴곡 & 내회전 또는 외회전 시켜 놓음
    • 손목, 손 : 환자를 둥근의자에 앉히고 작은 테이블 위에 손을 올리고, 길고 얇은 배게를 손목 아래에 받침
    • 발 : 환자를 침대 바깥으로 발목 이하가 나오도록 눕힘

4.2 검사 및 시술 중

  • 이재현 pf.
    • 검사 중 기록 보조
      • 예문 형식 활용
    • 시술 결정 시, 해당 medication 준비(3-4 Regimen 참고)
    • 시술 보조
      • needle insertion 부위 포비돈 스틱으로 6회 swap
      • needle uncap 하여 시술 손에 전달
      • 시술 중 손이 없을 때 보조
        • 동영상 저장, depth, focus, power doppler전환 보조
  • 이병주 pf.

4.3 시술 후

  • needle insertion 부위 밴드 혹은 압박드레싱, 소독약 닦기
  • End exam 누르고 pacs에 저장
  • 환복

4.4 Regimen(이재현 pf.)

  • tamceton 혹은 dexa로 약을 준비하고, 나머지는 lidocaine을 1% 농도로 준비한다고 생각하면 됩니다.
  • 어깨
  • 무릎
  • 고관절
  • 팔꿈치, 손, 발
  • 척추(USG)
  • Calcium barbotage
  • IA : 20cc syringe + 22G spinal needle
    • 20cc : lido 2% 4cc + N/S 15cc + tam 1v
    • 12cc : (체구 작은 여성 or 심한contracture or articular side tear 등의 경우) lido 2% 2cc + N/S 10cc + tam1v
  • Bursa : 10cc syringe + 25G 정림 needle
    • 10cc : lido 2% 2cc + N/S 7cc + tam 1v
  • IA : 어깨와 같음
  • Aspiration : 20cc syringe 빈 것 하나 더 추가
    • aspiration 양 / 색깔 check / 필요시 검체나가기
  • IA synovian : 23G needle 이상 사용하기
  • IA, Bursa : 어깨 bursa와 같음
  • CET/CFT : 5cc syringe + 25G정림 needle : Ropiva 2cc + N/S 2cc + tam 1v
  • Small tendon sheath / bursa / joint : 3cc syringe + 25G needle : Ropiva 1cc + N/S 1.5cc + tam 0.5v (26G 짧지 않은 바늘 있는지 확인)
  • TF : 3cc syringe + 25G 정림 needle : lido 1cc + N/S 2cc + tam 1v
  • CTS : 3cc syringe + 25G 정림 needle : N/S 2cc + tam 1v (리도카인에 마비감호소)
  • Facet / MBB : 10cc syringe + 25G 정림 needle : N/S 5cc + triam 1@ (양측이면 2개 준비)
  • Root block : 5cc syringe + 22G spinal needle : Ropiva 0.5 cc + N/S 3.5cc + tam 1v (cervical은 dexa 2@)
  1. 3cc syringe + 25G 정림 needle : lido 1% 3cc
  2. 10cc 루어락 syringe 3~10개 + 18G spinal needle 1개
    1. 첫 3개는 lido 1%로 9cc
    2. 석회쇄척 잘되면 near clear lavage될 때까지 N/S 9cc 로 계속 만들기
  3. 3cc syringe : N/S 2cc + tam 1v (약 넣자고 하면 N/S syringe 만들기 중단하고 제조)
  4. 끝나면 거즈로 압박드레싱
  5. 석회나온 시린지는 어두운 배경에 세워서 석회침전시키고 사진찍어 pacs에 등록
