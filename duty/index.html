1. 기간 · 공휴일 · 수동 인원

신정·3.1절·어린이날·현충일·광복절·개천절·한글날·성탄절은 자동 인식. 음력/대체공휴일은 직접 추가하세요. · 수동 배정 인원은 자동 배정에서 제외되고, 생성 후 일정표에서 원하는 날에 직접 넣습니다.

2. 당직자 (자동 배정)

이름불가일 (예: 3, 5, 10-12)평일이월주말이월
이월값은 보통 0으로 두고, 생성 후 "다음 달 준비"가 자동으로 채웁니다.
